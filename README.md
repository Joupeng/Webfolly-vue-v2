# 網中愚｜你的每次點擊都是改變媒體環境的契機
Try to solve the problem of clickbait and fake news.

### Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### To browse the contents of the dist folder

```
npm run preview
```
